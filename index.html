ChatUI Tutorial

Installing the mod

First download ChatUI from Here.
Then install it via the upload section of BMBF.
Make sure too enable it in mods tab if you haven't already.

Finding Your Config

The config file can be found at Android/data/com.beatgames.beatsaber/files/mod_cfgs/
Here we can modify our settings like blacklisted people (if someone is annoying or to blacklist bots)

Editing The Config

With the Config file we have we will see something similar too This. Here we can find the rotation and position values for where the chat is in game or in the menu(im sure you can figure this out later.).Then choose your channel name.

Uploading the Config back onto the quest

This is pretty simple open up side quest then maneuver back to Android/data/com.beatgames.beatsaber/files/mod_cfgs/ And drag the .json file to the top left corner of sidequest then restart Beatsaber

(IMPORTANT: Make sure that the file is named EXACTLY the same, so no ChatUI (1).json)
